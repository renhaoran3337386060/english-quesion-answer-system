# 打包说明

## 一、Windows EXE 打包

### 方法一：一键打包（最简单）
```bash
双击运行：一键打包.bat
```

### 方法二：命令行打包
```bash
python build_exe.py
```

### 输出文件
- `dist\英语练习工具.exe` - 单文件版
- `英语练习工具-便携版\` - 便携版文件夹

---

## 二、Android APK 打包

### 方法一：一键打包（推荐）
```bash
双击运行：一键打包APK.bat
```

### 方法二：命令行打包
```bash
python build_apk.py
```

### 方法三：Docker打包
```bash
# Windows
docker-build.bat

# Linux/Mac
./docker-build.sh
```

### 方法四：GitHub Actions云打包
1. 推送代码到GitHub
2. 进入 Actions → Build Android APK
3. 点击 Run workflow

---

## 四、环境要求

### Windows EXE
- Python 3.8+
- PyInstaller (`pip install pyinstaller`)

### Android APK
- Docker Desktop (推荐)
- 或 WSL2 + Ubuntu
- 或 Linux系统
- 磁盘空间 20GB+

## 五、常见问题

### Q: 打包失败怎么办？
A: 检查以下几点：
1. 依赖是否安装完整
2. 网络连接是否正常
3. 磁盘空间是否充足
4. 查看详细错误日志

### Q: APK安装后闪退？
A: 可能原因：
1. Android版本过低（需要5.0+）
2. 缺少权限
3. 查看logcat日志

### Q: 如何减小APK体积？
A: 使用release模式打包：
```bash
buildozer android release
```
